# Personal Portfolio

A modern and fully responsive developer portfolio built using **Next.js** with **Tailwind CSS**. This portfolio highlights my professional experience, projects, skills, and achievements with a clean UI and smooth animations. Designed for accessibility, the site provides a polished and interactive overview of my work as a developer and my passion.

## Tech Stack

- **Framework :** Next.js + React
- **Styling :** Tailwind CSS
- **Fonts :** Google Fonts
